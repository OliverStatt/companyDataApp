# companyDataApp
Die App zeigt Gewinn und Bilanzsumme bzw. Umsatz zu allen deutschen Unternehmen an, die diese Zahlen veröffentlicht haben.

## Live Demo:
https://gewinnzahlen.herokuapp.com

## Usage:
Firmennamen eingeben und auf suchen klicken (einige Firmen haben keine Zahlen veröffentlicht).

## API:
https://gewinnzahlen.herokuapp.com/api
